上传控件整合: fileUploader
图片上传控件：imgUploader
图片编辑器功能支持浏览器有Chrome,firefox,IE10以及IE10以上 ，如果您使用的浏览器有极速模式，请开启！
$('#carImages').fileUploader({
	uploaderType: 'imgUploader',
	itemWidth: 120,
	itemHeight: 70,
	fileNumLimit: 2,
	fileSingleSizeLimit: 2*1024*1024, /*2M*/
	fileVal: 'myFiles',
	inputName: 'carImages',
	server: base + '/member/ajaxImageUpload',
	toolbar:{
		edit: {
			width:480,
			height:280
		},
		move: true,
		del: true
	}
});

$('#etcImages').fileUploader({

参数名默认值介绍
uploaderType无默认值，请设置(imgUploader:图片上传控件)上传控件UI选择
itemWidth(imgUploader的情况下，无默认值，请设置)单个上传单位隐藏input所在区域的宽
itemHeight(imgUploader的情况下，无默认值，请设置)单个上传单位隐藏input所在区域的高
fileNumLimit无默认值，请设置上传文件数量限制
fileSingleSizeLimit无默认值，请设置单个文件大小限制
inputName无默认值，请设置表单中隐藏input的name
fileVal无默认值，请设置文件上传时的name
server无默认值，请设置后台处理上传请求的URL
toolbar对象，默认值{edit:{width:400,height:400},move:true,del:true}工具栏配置：edit配置裁切画布大小，move配置左右移动，del配置删除，设置为false是关闭功能
